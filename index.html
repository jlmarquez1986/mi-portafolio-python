FacturaX

El proyecto está diseñado como una solución inicial para pequeñas empresas o autónomos. Se trata de un sistema de facturación, una aplicación sencilla, visual e intuitiva que permite a los usuarios gestionar clientes, productos, servicios y generar facturas en PDF.

Tecnologías usadas:

  • Lenguaje: Python
  • Framework de Interfaz Gráfica (GUI): Tkinter y ttkbootstrap.
  • Base de Datos: SQLite

Librerías Adicionales:

  • bcrypt (para la seguridad de contraseñas)
  • reportlab (para generar los PDFs)
